# Getting Listed on GitHub Topics

We use GitHub topics to list all projects tagged with `tricycler-stack`. Any public repo with that topic appears automatically at [github.com/topics/tricycler-stack](https://github.com/topics/tricycler-stack).

## Add the topic to your repo

1. Go to your repository on GitHub
2. Click the gear icon next to **About**
3. Under **Topics**, type `tricycler-stack` and press Enter
4. Click **Save changes**

Your stack is now discoverable by anyone looking for tricycler projects.

## Make your listing useful

GitHub shows your repo name, description, stars, and last update. Visitors read your description first — make it one clear sentence describing your stack.
